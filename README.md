# Frontend Project

A clean, responsive single-page website built with **pure HTML5, CSS3, and vanilla JavaScript** — zero dependencies, zero build steps.

## Live preview

Open `index.html` directly in your browser — no server required.

## Project structure

```
frontend_project/
├── index.html          # Main HTML entry point
├── css/
│   └── styles.css      # All styles (custom properties, responsive layout)
├── js/
│   └── script.js       # Interactive behaviour (theme, nav, counters, form)
└── assets/             # Images and other static assets
```

## Features

| Feature | Description |
|---|---|
| **Responsive layout** | Mobile-first design, tested from 320 px to 1440 px |
| **Dark / light mode** | Toggle persisted in `localStorage`; respects `prefers-color-scheme` |
| **Smooth navigation** | Scroll-spy highlights the active section link |
| **Animated counters** | Stats count up when they scroll into view |
| **Contact form** | Client-side validation with accessible error messages |
| **Zero dependencies** | No npm, no bundler — just open and go |

## Getting started

1. Clone or download the repository.
2. Open `index.html` in any modern browser.

```bash
git clone https://github.com/SayanGhosh-dot/frontend_project.git
cd frontend_project
# Then open index.html in your browser
```

## Browser support

Works in all evergreen browsers: Chrome, Firefox, Safari, Edge.
