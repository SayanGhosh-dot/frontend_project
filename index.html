About

A brief introduction to this project

This is a modern frontend project built using vanilla HTML5, CSS3, and JavaScript. It demonstrates best practices for structuring a web project without relying on any external frameworks.

The project features a fully responsive layout, smooth scrolling navigation, an interactive theme toggle, and clean semantic markup that is accessible and SEO-friendly.
